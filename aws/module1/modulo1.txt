| Servicio                               | Tipo           | Descripción                                                                                |
| -------------------------------------- | -------------- | ------------------------------------------------------------------------------------------ |
| **EC2** (Elastic Compute Cloud)        | Cómputo        | Servidores virtuales (VMs) bajo demanda. Puedes instalar cualquier software.               |
| **S3** (Simple Storage Service)        | Almacenamiento | Guarda archivos y objetos. Ideal para imágenes, backups y contenido estático.              |
| **IAM** (Identity & Access Management) | Seguridad      | Controla usuarios, roles y permisos. Define qué recursos puede usar cada uno.              |
| **Lambda**                             | Serverless     | Ejecuta código sin servidores. Pagas solo por ejecución. Ideal para microservicios y APIs. |
| **RDS** (Relational Database Service)  | Bases de datos | Proporciona bases de datos administradas como MySQL, PostgreSQL o SQL Server.              |
