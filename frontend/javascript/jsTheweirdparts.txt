🔵 MÓDULO 5 — Closures (nivel motor)

Closure no es magia.

Es:

Una función + referencia al environment donde fue creada.

Qué guarda el engine

Function object

Hidden [[Environment]]

Reference al lexical scope

Ejemplo:

function outer() {
  let counter = 0;
  return function inner() {
    counter++;
    console.log(counter);
  };
}

const fn = outer();
fn();
fn();

Internamente:

outer sale del stack

Pero environment NO se elimina

Permanece en heap
